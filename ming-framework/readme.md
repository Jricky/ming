### ming-framework

`ming项目SSM框架公共模块
`
### 包结构

``` lua
ming.framework
├── constant        基础常量包
├── core            核心架构
    ├── controller  控制层入库基类
    ├── dao         数据层接口
    ├── po          po实体类
    ├── service     基础业务层
        ├── cache   初始化缓存
    ├── vo          vo基类
├── exception       异常处理
├── util            工具包
```
